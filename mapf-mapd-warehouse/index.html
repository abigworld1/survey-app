  • CADENCE: Predicting Realized MAPF Execution Time Beyond Sum of Costs
    2026-06-03 ・ 従来のMAPF（Multi-Agent Path Finding）におけるSum of Costs (SoC) は、物理ロボットの実際の実行時間（wall-clock time）を正確に予測できない。本研究は、SoCに加えて「原始的な運動負荷（primitive motion burden）」と「相互作用を考慮した調整構造（interaction-aware coordination structure）」を導入するCADENCEを提案し、7台の差動駆動ロボットを用いた実機実験により、特に運動負荷の追加が予測精度を大幅に向上させることを示した。
  • STEAM: A Training-Free Congestion-Aware Enhancement Framework for Decentralized Multi-Agent Path Finding

  • Priority Inheritance with Backtracking for Iterative Multi-agent Path Finding
    2019-01-31 ・ Lifelong MAPF/MAPDを解決するための、優先度継承（Priority Inheritance）とバックトラッキングを組み合わせた新しい劣最適アルゴリズム「PIBT」を提案する。PIBTは、低優先度エージェントが移動を妨げる場合に高優先度エージェントの優先度を一時的に継承させることでデッドロックを防ぎ、有効な移動先がない場合にバックトラッキングを行うことで、大規模なエージェント数でも高速かつ安全な経路生成を可能にする。
